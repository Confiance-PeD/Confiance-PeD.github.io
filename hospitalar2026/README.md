# Hospitalar 2026 — Sales App

App de apoio comercial para a feira Hospitalar 2026.  
Acesse em: **https://SEU-USUARIO.github.io/hospitalar2026/**

## Funcionalidades

- 📊 Dashboard com métricas de leads e vendas
- 🔬 Funil de visitantes da feira
- 💼 Pipeline público em andamento
- ⚡ JOLT — Análise de objeções com IA (Claude.ai)
- 📁 Documentos e materiais de apoio
- 💰 Tabela de preços Hospitalar 2026

## Como usar localmente

Basta abrir o arquivo `index.html` no navegador. Não requer servidor.

## Deploy

Hospedado via **GitHub Pages** — branch `main`, pasta raiz.

## Tecnologias

- HTML/CSS/JS puro (sem dependências de build)
- localStorage para persistência de dados
- Tabler Icons (CDN)
- Integração com Claude.ai para análise JOLT
